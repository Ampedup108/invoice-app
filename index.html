Field Invoice Manager

Sign in with Google to sync your clients, hours, and invoices to Google Sheets.

Your data is stored in your own Google Sheet — nothing is stored on any server.

Setup: Host on GitHub Pages

This app must be served from a real URL for Google Sign-In to work. The free option is GitHub Pages:

  1. Go to github.com → create free account
  2. New repository → name it invoice-app → Public → Create
  3. Upload this HTML file renamed to index.html
  4. Settings → Pages → Source: main branch → Save
  5. Your URL will be: https://USERNAME.github.io/invoice-app
  6. In Google Cloud Console → Credentials → add that URL to Authorized JavaScript origins
  7. Open that URL in Safari → Sign in with Google

On iPhone: tap Share → Add to Home Screen for a full app experience.
